jQuery

jQuery is a popular open-source JavaScript library that simplifies and streamlines the process of developing dynamic and interactive web pages. It was first released in 2006 by John Resig and has since become one of the most widely used JavaScript libraries, with a large and active community of developers and contributors.

jQuery provides a powerful and easy-to-use API that allows developers to manipulate HTML documents, handle events, create animations, and interact with server-side data using AJAX. Its syntax is designed to be concise and intuitive, making it easier to write complex JavaScript code with less effort.


Some of the key features of jQuery include:


  • DOM manipulation: jQuery provides a simple and powerful way to manipulate the Document Object Model (DOM) of an HTML document. This includes selecting, adding, removing, and modifying HTML elements, as well as changing their attributes and properties.
    • To select an HTML element and change its text, you can use the following code: 
    
// Select the element with ID "myElement" and change its text
$("#myElement").text("New text");
    To add a new element to the page, you can use the following code: 
    
// Create a new <p> element and append it to the <body> element
var newElement = $("<p>").text("This is a new paragraph.");
$("body").append(newElement);
  • Event handling: jQuery simplifies the process of handling user events, such as mouse clicks and keyboard inputs. It provides a consistent and cross-browser-compatible way to attach event handlers to HTML elements and handle events in a consistent manner.
    • To handle a click event on a button, you can use the following code: 
    
// Attach a click event handler to the button with ID "myButton"
$("#myButton").click(function() {
  alert("Button clicked!");
});
    To handle a key press event on a text input, you can use the following code: 
    
// Attach a key press event handler to the text input with ID "myInput"
$("#myInput").keypress(function(event) {
  console.log("Key pressed: " + event.which);
});
  • AJAX: jQuery provides a simple and powerful API for making asynchronous HTTP requests to a server and handling the response. This allows developers to build dynamic and interactive web pages that can update their content without requiring a full page reload.
    • To make an AJAX request to a server and update the page with the response, you can use the following code: 
    
// Make an AJAX request to the server and update the <div> element with ID "myDiv" with the response
$.ajax({
  url: "/api/data",
  type: "GET",
  success: function(data) {
    $("#myDiv").text(data);
  }
});
  • Animations: jQuery provides a powerful and easy-to-use animation API that allows developers to create complex and visually appealing animations with minimal effort. This includes animating HTML elements, CSS properties, and even entire HTML documents.
    • To animate an HTML element, you can use the following code: 
    
// Animate the element with ID "myElement" to fade out over 1 second
$("#myElement").fadeOut(1000);
  • Plugins: jQuery has a large and active community of developers who have created a wide range of plugins and extensions that extend the functionality of the library. These plugins can be easily integrated into any jQuery project, allowing developers to add new features and functionality with minimal effort.
    • To use a jQuery plugin, you can simply include its script file and use its API. For example, to use the jQuery UI library to create a draggable element, you can use the following code: 
    
// Include the jQuery UI script file
<script src="https://code.jquery.com/ui/1.12.1/jquery-ui.js"></script>

// Make the element with ID "myElement" draggable using the jQuery UI library
$("#myElement").draggable();
    These are just a few examples of the many things you can do with jQuery. With its simple and intuitive API, you can easily manipulate the DOM, handle events, make AJAX requests, create animations, and use plugins to extend its functionality.

Overall, jQuery is a powerful and versatile library that has revolutionized the way web developers create dynamic and interactive web pages. Its simplicity and ease of use have made it a favorite among developers of all skill levels, and its active community ensures that it will continue to evolve and improve for years to come.

jQuery Syntax

  
$(selector).action()

  • $ sign to define jQuery
  • (selector) to "query (or find)" HTML elements
  • jQuery action() to be performed on the element(s)

Document Ready Event

        
            $(document).ready(function(){
                // jQuery methods go here...
            })

All jQuery methods are inside ready event, it's because it may be possible the image hasn't been loaded yet and you are trying to get image size or trying to hide an element. Then you will get error.

Shorter method for document ready event: 

        
           $(function(){
              // jQuery methods go here ....

           })

Selectors

Selectors allow you to select and manipulate HTML elements.

$(*) -> Selects all elements
$(this) -> Selects the current HTML element
$("p:first") -> Selects the first <p> element
